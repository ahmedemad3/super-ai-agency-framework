# Agency Enterprise Preset

This preset customizes Spec-Kit for the Super AI Agency Framework:

- **Specifications** → Business Requirements Document (BRD)
- **Plans** → Software Architecture Document (SAD)
- **Tasks** → Sprint Backlog with `[P]` parallel markers
- **Constitution** → 8-section governance document

## How It Works

This preset overrides Spec-Kit's default artifact paths to point to the Agency's `Artifacts/` directory structure instead of `.specify/specs/`. The Agency's `Artifacts/` folder is the source of truth.
